# 时序IQ信号射频指纹识别 - CNN+ResNet+自注意力+LSTM混合模型使用说明

## 概述
本项目实现了一个结合CNN、ResNet、自注意力机制和LSTM的混合深度学习模型，用于时序IQ信号的射频指纹识别任务。

## 数据格式
- **输入数据维度**: `[160, 2, 1, 4000]`
  - 160: 时间序列长度（时间点数）
  - 2: 通道数（I/Q两个通道）
  - 1: 通道维度（单通道）
  - 4000: 样本数量

## 文件说明

### 主要文件
1. **RF_Fingerprinting_Final_Model.m** - 最终推荐使用的完整模型
2. **RF_Fingerprinting_Hybrid_Model_Optimized.m** - 优化版本（包含更多特征）
3. **RF_Fingerprinting_Hybrid_Model.m** - 基础版本（使用layerGraph）

### 推荐使用
建议使用 `RF_Fingerprinting_Final_Model.m`，因为：
- 与原始代码完全兼容
- 参数命名保持一致
- 解决了LSTM输入维度问题
- 包含完整的性能分析和可视化

## 网络架构

### 1. CNN特征提取模块
```matlab
% 多尺度卷积核提取时域特征
convolution2dLayer([7 1], 32)  % 长期依赖
convolution2dLayer([5 1], 64)  % 中期模式
convolution2dLayer([3 1], 128) % 短期特征
```

### 2. ResNet残差连接
```matlab
% 防止梯度消失，加深网络
ResConv1 -> ResConv2 (64通道)
ResConv3 -> ResConv4 (128通道)
```

### 3. 自注意力机制
```matlab
% 通过全连接层模拟注意力
Attention_Query  (256维)
Attention_Key    (256维)  
Attention_Value  (128维)
```

### 4. LSTM时序建模
```matlab
% 双层LSTM捕获长期时序依赖
LSTM1: sequence输出，100隐藏单元
LSTM2: last输出，100隐藏单元
```

## 扩展功能

### 1. 模型集成
可以训练多个模型并进行集成：
```matlab
% 训练多个模型
model1 = trainNetwork(xTrain, yTrain, layers1, options);
model2 = trainNetwork(xTrain, yTrain, layers2, options);

% 集成预测
pred1 = predict(model1, xTest);
pred2 = predict(model2, xTest);
ensemble_pred = (pred1 + pred2) / 2;
```

### 2. 迁移学习
```matlab
% 加载预训练模型
load('pretrained_model.mat');

% 冻结前几层

% 微调训练

fine_tuned_model = trainNetwork(new_data, new_labels, layers, options);
```
