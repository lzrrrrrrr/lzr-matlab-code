# 改进的射频指纹识别系统

## 项目概述

这是一个基于深度学习的射频指纹识别系统，专门针对WLAN设备的IQ信号进行身份识别。本项目在原有LSTM模型基础上，设计了一个结合CNN、ResNet、自注意力机制和LSTM的混合深度学习模型，显著提高了识别准确率。

## 主要改进

### 1. 混合深度学习架构
- **CNN特征提取层**: 使用多尺度卷积核（9×2, 7×2, 5×2, 3×2）提取IQ信号的局部特征
- **ResNet残差连接**: 实现跳跃连接，解决深度网络梯度消失问题，提高特征学习能力
- **自注意力机制**: 通过注意力权重突出重要特征，提高模型对关键信息的关注度
- **双向LSTM**: 处理时序信息，捕获IQ信号的时间依赖关系

### 2. 模型优化策略
- **批量归一化**: 加速训练收敛，提高模型稳定性
- **Dropout正则化**: 防止过拟合，提高泛化能力
- **全局平均池化**: 减少参数量，提高计算效率
- **梯度裁剪**: 防止梯度爆炸，确保训练稳定

### 3. 训练参数优化
- **学习率调度**: 采用分段衰减策略，初始学习率0.0005，每8轮衰减至0.2倍
- **早停机制**: 验证耐心12轮，防止过拟合
- **数据增强**: 支持IQ数据的噪声添加和相位旋转增强
- **批次大小优化**: 使用128的小批次提高训练稳定性

### 模型架构特点

#### 网络结构
```
输入层 [160×2×1] (IQ信号)
    ↓
CNN特征提取块1 (Conv 9×2, 64 filters)
    ↓
CNN特征提取块2 (Conv 7×2, 128 filters) + 残差连接
    ↓
CNN特征提取块3 (Conv 5×2, 256 filters) + 残差连接
    ↓
CNN特征提取块4 (Conv 3×2, 512 filters)
    ↓
全局平均池化
    ↓
自注意力机制 (FC 512 + Sigmoid权重)
    ↓
双向LSTM (256 → 128 隐藏单元)
    ↓
全连接分类层 (512 → 256 → numClasses)
    ↓
输出 (射频指纹类别)
```

#### 关键改进点
1. **多尺度特征提取**: 大卷积核捕获长距离依赖关系
2. **残差学习**: 跳跃连接提高梯度流动
3. **注意力权重**: 动态调整特征重要性
4. **时序建模**: BiLSTM处理信号时间特性

## 技术细节

### 数据预处理
- Z-score标准化输入数据
- IQ信号分离为实部和虚部
- 数据维度重塑为[160×2×1×N]

### 网络训练
- 优化器: Adam
- 最大轮次: 40
- 验证频率: 每1/3轮次验证一次
- L2正则化: 0.0001

## 故障排除

### 常见问题

1. **内存不足**
   - 减小`miniBatchSize`参数
   - 降低网络复杂度

2. **训练速度慢**
   - 使用GPU训练（修改`ExecutionEnvironment`为'gpu'）
   - 减少训练轮次

3. **准确率不达标**
   - 增加训练数据
   - 调整学习率参数
   - 增加网络深度

4. **网络结构错误**
   - 代码会自动回退到简化模型
   - 检查MATLAB工具箱版本

### 性能优化建议

1. **数据层面**
   - 增加训练样本数量
   - 使用数据增强技术
   - 平衡各类别样本数量

2. **模型层面**
   - 调整网络深度和宽度
   - 优化正则化参数
   - 尝试不同的激活函数

3. **训练层面**
   - 使用学习率预热
   - 实施余弦退火调度
   - 增加验证数据比例
